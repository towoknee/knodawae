Most useful programs require us to be able to examine certain conditions, and be able to proceed with our program based on that. There are three main conditional statements that Python uses: if, elif, and else. The statement after the conditional is called the condition (x > 0, x < 0). If the conditional statement turns out to be true, then the resulting lines of code in the condition is executed. If it's not true, then it proceeds to the next conditional statement and then to the next one until it is true. In our example, the coin would be placed in the else statement since it is not greater than zero, nor is it less than zero. Else is everything else that the prior conditional statements do not cover. Here are some examples:

if x % 2 == 0:
  print ("your number is even")
else:
  print ("your number is odd")

If you are given x = 4, 4 % 2 would be 0, which would output "your number is even"

An extremely important topic in programming in general is iteration. There are two types of iterators in Python: for and while. A for loop iterates through a given sequence. It can be best seen through an example. Say we are given a list of numbers list = [1, 2, 4, 5]. In order to go through this list (the list is assigned to the variable list) and output the results, the following syntax would look something like this:

for x in list:
  print(x)

This would go through the whole list and print out all of the values. x would represent each element in the list. The output would be 1, 2, 4, 5. We also do not have to specify the list and can use range. range(4) would be the same as [0,1,2,3] and range(3,6) would be [3,4,5]. So:

for x in range(4):
  print (x)

Would result in 0,1,2,3

A while loop repeats the content of its body as long as a certain condition is met. Its syntax looks a little something like this:

while x > 4:
  print ("Likey Likey")

However, many of you intellectuals out there may have noticed that there is an error to this. There must be a statement that continuously updates the variable that is being compared (x). If there isn't one, then this loop would go on forever. This is called an infinite loop. You don't want this. The correct code would be:

while x>20:
  print ("Likey Likey")
  x = x - 1

The last statement would let the conditional in the while statement become false at some point, thus ending the loop. We can also utilize the break function. This causes the loop to immediately (who would have thought?) break out of the loop. Therefore, combining an if statement with a while statement can be extremely useful in many situations. For ex:

while age < 21:
  print ("cool")
  if age == 13:
    break
  age = age + 1

Here, if the age is 13, then the loop ends.
